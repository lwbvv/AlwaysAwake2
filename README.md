# AlwaysAwake2

_일을 효율적으로 할 수 있도록 계획을 세우고 습관을 들이는데 도움을 주는 어플리케이션입니다. 새해에 못 지켰던 계획이나 습관을 들여 봅시다_

## 사용 기술
+ Java, Google Map, Google Place Autocomplete




## 프로젝트 기간
2019.05 ~ 2019.07
